# Module 2: The Digital Twin (Gazebo & Unity)

Welcome to the second module of The Autonomous Humanoid curriculum. This module focuses on creating digital twins using Gazebo and Unity for humanoid robotics applications.

## What You'll Learn

In this module, you'll explore:
- Physics simulation and collision detection
- Rendering techniques in Unity
- Sensor simulation including LiDAR and IMU

## Prerequisites

Before starting this module, ensure you have:
- Completed Module 1
- Basic understanding of 3D graphics concepts

## Getting Started

Begin with the first chapter on [Physics and Collisions](./physics-collisions.md) to understand the fundamentals of physics simulation.
