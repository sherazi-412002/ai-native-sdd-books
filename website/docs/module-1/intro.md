# Module 1: The Robotic Nervous System (ROS 2)

Welcome to the first module of The Autonomous Humanoid curriculum. This module introduces you to the fundamental concepts of ROS 2, which serves as the nervous system for autonomous humanoid robots.

## What You'll Learn

In this module, you'll explore:
- The core concepts of ROS 2 (nodes, topics, services)
- How to bridge different programming languages with rclpy
- Creating humanoid robot models with URDF

## Prerequisites

Before starting this module, ensure you have:
- Basic programming knowledge
- Understanding of robotics concepts

## Getting Started

Begin with the first chapter on [Nodes, Topics, and Services](./nodes-topics-services.md) to understand the foundational concepts of ROS 2.
